# FundusClassifier EXE 빌드 및 사용 방법

---

## 📁 사전 준비 — 파일 구성

빌드 전, 아래 파일들이 **같은 폴더**에 있어야 합니다.

```
📁 프로젝트 폴더
├── app.py
├── build_exe.py
├── fold1_best.pth
├── fold2_best.pth
└── fold3_best.pth
```

---

## ⚙️ 1단계 — 필수 패키지 설치

터미널(CMD 또는 PowerShell)을 열고 아래 명령어를 순서대로 실행하세요.

### PyTorch (CPU 전용 버전)
```bash
pip uninstall torch torchvision -y
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

> ⚠️ CUDA 버전 PyTorch가 설치되어 있으면 EXE 크기가 매우 커집니다. 반드시 CPU 전용으로 교체하세요.

### 나머지 패키지
```bash
pip install pywebview opencv-python matplotlib pillow pyinstaller
```

---

## 🔨 2단계 — EXE 빌드

프로젝트 폴더에서 아래 명령어를 실행합니다.

```bash
python build_exe.py
```

- 빌드는 **약 5~15분** 소요됩니다.
- 완료되면 `dist/FundusClassifier/` 폴더가 생성됩니다.
- 가중치 파일(`.pth`) 3개가 EXE 안에 자동으로 포함되므로 **별도 전달이 필요 없습니다.**

---

## 📦 3단계 — 배포

빌드가 완료되면 아래 폴더를 zip으로 묶어 전달합니다.

```
dist/
└── FundusClassifier/   ← 이 폴더 전체를 zip으로 압축
    ├── FundusClassifier.exe
    ├── _internal/
    └── ...
```

받는 사람은 zip을 풀고 `FundusClassifier.exe`를 실행하기만 하면 됩니다.

---

## 🖥️ 앱 사용 방법

### 1. 실행
`FundusClassifier.exe`를 더블클릭하여 실행합니다.
앱이 시작되면 가중치 파일을 **자동으로 로드**합니다.

### 2. 이미지 업로드
- `이미지 선택` 버튼을 클릭합니다.
- **좌안 + 우안이 합성된 이미지** (jpg / png) 를 선택합니다.

### 3. 분류 실행
- `🔍 분류 + Grad-CAM 실행` 버튼을 클릭합니다.
- 잠시 후 분류 결과와 Grad-CAM 히트맵이 표시됩니다.

### 4. 결과 확인
| 항목 | 설명 |
|------|------|
| 분류 결과 | Normal / NAION / ON 중 하나 |
| 확률 바 | 각 클래스별 예측 확률 |
| Grad-CAM 히트맵 | AI가 판단 근거로 삼은 영역 시각화 (붉을수록 중요) |

### 5. 결과 저장 / 복사
- `클립보드 복사` — 결과 텍스트를 클립보드에 복사
- `PNG 결과 저장` — Grad-CAM 히트맵 이미지를 PNG로 저장

---

## ❗ 주의사항

- 본 결과는 **AI 보조 분석 결과**이며, 의사의 최종 임상 판단을 대체하지 않습니다.
- 확진을 위해 OCT, 시야 검사 등 추가 검사가 필요할 수 있습니다.
- 입력 이미지는 반드시 **좌안 + 우안 합성 이미지**여야 합니다.

---

## 🛠️ 문제 해결

| 증상 | 해결 방법 |
|------|----------|
| `No module named 'webview'` | `pip install pywebview` 실행 |
| `No module named 'torch'` | PyTorch 설치 명령어 재실행 |
| 빌드 실패 | 터미널 오류 메시지 확인 후 누락 패키지 설치 |
| 앱 실행 후 모델 로드 실패 | 빌드 시 `.pth` 파일 3개가 같은 폴더에 있었는지 확인 |
